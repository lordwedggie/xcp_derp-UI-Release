2. Strip excluded files: `git rm --cached js/motha/themeManagerV2.js js/motha/themeManagerV2_core.js python/derpThemeManagerV2.py` and `git rm --cached -r .deepseek/`
3. Ensure `.github/workflows/publish_action.yml` is present
4. Commit with message "release vX.Y.Z"
5. Create tag: `git tag -a vX.Y.Z -m "vX.Y.Z"`
6. Push release branch: `git push public daily-development:refs/heads/release/vX.Y.Z`
7. Push tag: `git push public vX.Y.Z`
8. Update main: `git push --force-with-lease public daily-development:main`
9. Restore excluded files: `git add js/motha/themeManagerV2.js js/motha/themeManagerV2_core.js python/derpThemeManagerV2.py .deepseek/` and commit "restore excluded files to daily-development"
10. Push to origin: `git push --force-with-lease origin daily-development`

The release repo auto-publishes to Comfy Registry via `.github/workflows/publish_action.yml` which triggers on push to `main` when `pyproject.toml` changes.
