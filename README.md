xcpDerpNodes by lordwedggie

xcpDerpNodes is a custom node pack for ComfyUI with derpy loaders, palettes, dockable panels, prompt tools, and workflow utilities.

It is built for people who want their workflow to be useful, weird, organized, and just a little bit unhinged.

## English

xcpDerpNodes bundles a growing pile of custom tools for ComfyUI, including:

- dockable UI panels
- palette and theme helpers
- prompt and text editing tools
- loaders and workflow utilities
- assorted quality-of-life nonsense that somehow became useful

This pack is for anyone who wants their node graph to feel less like a warehouse accident and more like a carefully managed goblin command center.

`xcpDerpNodes` is a custom node pack for ComfyUI focused on building a more interactive, more visual, and slightly more derp workflow experience.

It mixes utility nodes, loader nodes, dockable UI panels, wireless-style helper flows, and a shared UI framework that lets parts of the node graph behave more like tiny tools instead of plain boxes with sockets.

## What This Pack Is

This project is built around a few core ideas:

- make complex node workflows easier to inspect and control
- provide richer in-node UI for editing, browsing, previewing, and organizing data
- support more modular workflow building through helper nodes and wireless signal patterns
- make the graph feel less static and more like an actual workspace

In other words: it is trying very hard to make ComfyUI a little smarter, a little cleaner, and a little more gloriously over-engineered.

## What You Can Expect

Depending on the branch and release, this repository may include things like:

- loader-style nodes for common workflow selections
- dockable and collapsible UI-driven nodes
- LoRA helper tools and preview/detail panels
- prompt editing helpers
- image and signal handling utilities
- shared UI systems used across the custom nodes

## Installation

Clone or copy this repository into your ComfyUI `custom_nodes` directory.

```bash
git clone https://github.com/lordwedggie/xcp_derpNodes.git
```

Then restart ComfyUI.

## Notes

- This project moves fast, so some branches may be more experimental than others.
- Release branches and tagged versions are the safest place to start if you want a more stable setup.
- If something behaves strangely, there is a decent chance it is either a bug or an advanced feature pretending to be one.
