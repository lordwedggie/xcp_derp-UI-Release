# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">调度器加载器</span>

![[derpSchedulerLoader_01.jpg]]
将调度器名称加载到节点面板上，一键切换。能不用下拉菜单就从 Normal 跳到 Karras，这才叫掌控人生。

<span style="color: #ffc680"><strong>重要：</strong></span> 此加载器需要 [[Management Nodes/Derp Router|derpRouter]] 才能工作。没有它，这玩意儿就是个默默鄙视你的调度器名单。

### <span style="color: #80ffc0">功能</span>

<span style="color: #80aaff"><strong>选择调度器下拉菜单</strong></span>：浏览并添加调度器到面板。内置搜索，毕竟在 Exponential、Karras、SGU 以及它们的十七个变体里滚来滚去可不是什么愉快体验。

<span style="color: #80aaff"><strong>刷新按钮</strong></span>：ComfyUI 构建里有新调度器了？点这个，别重启。

<span style="color: #80aaff"><strong>清空按钮</strong></span>：一键炸掉整个面板。空的时候灰色禁用，防止你点一个无事可干的按钮。

<span style="color: #80aaff"><strong>移除按钮</strong></span>：从面板中单独移除一个调度器。显示在激活的那个旁边，避免手滑删错。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：拖动调度器重新排列。功能上毫无意义，但你凌晨两点调的那些设置里有一半不也是这样。
