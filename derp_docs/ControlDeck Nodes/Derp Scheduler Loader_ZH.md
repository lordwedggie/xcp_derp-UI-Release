# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">调度器加载器</span>

![[derpSchedulerLoader_01.jpg]]
把调度器名字甩到面板上，点一下从 Normal 跳到 Karras。不用下拉菜单——这才叫掌控感。

<span style="color: #ffc680"><strong>重要：</strong></span> 得有 [[Management Nodes/Derp Router|derpRouter]] 才跑得起来。没它就是个默默鄙视你的调度器名单。

### <span style="color: #80ffc0">能干啥</span>

<span style="color: #80aaff"><strong>选调度器</strong></span>：挑一个加进面板。带搜索，毕竟 Exponential、Karras、SGU 还有它们那十七个变体，谁愿意一个个翻。

<span style="color: #80aaff"><strong>刷新</strong></span>：ComfyUI 更新了新调度器？点这个，别重启。

<span style="color: #80aaff"><strong>清空</strong></span>：一键炸了。空的时候灰着不让你点。

<span style="color: #80aaff"><strong>移除</strong></span>：踹走一个。挨着激活的显示，防止手滑。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：排着玩。功能上毫无意义，但你半夜两点调的那些设置有一半不也这样。
