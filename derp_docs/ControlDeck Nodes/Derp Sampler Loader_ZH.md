# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">采样器加载器</span>

![[derpSamplerLoader_01.jpg]]
可以放好多采样器在上面，点一下就能切！比如 Euler 和 DPM++ 2M 之间来回跳，超快。

<span style="color: #ffc680"><strong>重要：</strong></span> 你得先放一个 [[Management Nodes/Derp Router|derpRouter]] 在流程里，不然它不理你。

### <span style="color: #80ffc0">能干什么</span>

<span style="color: #80aaff"><strong>选择采样器</strong></span>：点开菜单挑一个加进来，还能打字搜索！不用在几十个名字里一个一个找，我视力不好，这个帮大忙了。

<span style="color: #80aaff"><strong>刷新</strong></span>：列表更新一下。

<span style="color: #80aaff"><strong>清空</strong></span>：一下子全删掉。但如果面板是空的，这个按钮是灰色的，因为没啥可删的嘛。

<span style="color: #80aaff"><strong>移除</strong></span>：只删一个。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：拖来拖去排着玩。
