# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">采样器加载器</span>

![[derpSamplerLoader_01.jpg]]
把采样器名字丢到面板上，点一下就能在 Euler 和 DPM++ 2M 之间横跳。人生苦短，别滚滚轮了。

<span style="color: #ffc680"><strong>重要：</strong></span> 得有 [[Management Nodes/Derp Router|derpRouter]] 才跑得起来。没它就是个态度很强硬的列表。

### <span style="color: #80ffc0">能干啥</span>

<span style="color: #80aaff"><strong>选采样器</strong></span>：挑一个加进面板。带搜索，不用眯着眼看10号字体的四十几个名字。

<span style="color: #80aaff"><strong>刷新</strong></span>：有新货了？点一下，比怀疑人生管用。

<span style="color: #80aaff"><strong>清空</strong></span>：一键扬了。面板空的时候灰着不让点——点个啥也干不了的按钮也太惨了。

<span style="color: #80aaff"><strong>移除</strong></span>：单独踹走一个。挨着激活的那个显示，省得手滑删错。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：随便拖。工作流没感觉，但你的强迫症会谢你。
