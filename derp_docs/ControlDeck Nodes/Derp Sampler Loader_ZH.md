# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">采样器加载器</span>

![[derpSamplerLoader_01.jpg]]
把采样器名字加载到面板上，点一下就能在 Euler 和 DPM++ 2M 之间横跳。

<span style="color: #ffc680"><strong>重要：</strong></span> 需要 [[Management Nodes/Derp Router|derpRouter]] 才能工作。

### <span style="color: #80ffc0">能干什么</span>

<span style="color: #80aaff"><strong>选择采样器</strong></span>：从下拉菜单浏览添加，支持搜索——不用眯着眼在四十几个名字里找。

<span style="color: #80aaff"><strong>刷新</strong></span>：更新采样器列表。

<span style="color: #80aaff"><strong>清空</strong></span>：一键清空面板。空的时候按钮禁用，毕竟点一个啥也干不了的按钮也太惨了。

<span style="color: #80aaff"><strong>移除</strong></span>：单独删除一个采样器。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：拖动条目调整顺序。
