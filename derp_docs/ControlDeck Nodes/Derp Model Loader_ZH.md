# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">模型加载器</span>

![[derpModelLoader_01.jpg]]
把 SDXL 模型甩到面板上，点一下就能切——告别下拉菜单，告别在两百多个叫 "最终版_v3_真的不改了" 的文件里扒拉。

<span style="color: #ffc680"><strong>重要：</strong></span> 得有 [[Management Nodes/Derp Router|derpRouter]] 这玩意儿才跑得起来。没它就是个花里胡哨的镇纸。

### <span style="color: #80ffc0">能干啥</span>

<span style="color: #80aaff"><strong>刷新</strong></span>：下了新模型？点一下，别跟原始人似的重启 ComfyUI。

<span style="color: #80aaff"><strong>移除</strong></span>：把模型踹出面板。它不记仇。大概。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：随便拖，随便排。对出图没影响，但治强迫症有奇效。

#### <span style="color: #80ffc0">系统面板里的玩意儿</span>

<span style="color: #80aaff"><strong>显示文件夹名</strong></span>：开关文件夹路径显示。关掉就没人知道你目录有多乱了。

<span style="color: #80aaff"><strong>切模型清显存</strong></span>：一换模型就倒垃圾。默认开着，ComfyUI 那内存管理已经够拉胯了，咱就别雪上加霜了。

<span style="color: #80aaff"><strong>加载配置</strong></span>：保存、读取、改名、复制你的面板排列——"人像专用"、"建筑专用"，爱咋分咋分。默认配置特意空着，因为你的模型散落在十几个文件夹里，除了你没人看得懂。
