# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">模型加载器</span>

![[derpModelLoader_01.jpg]]
这个东西可以把你的模型文件丢到一个面板上，然后你点一下就能换模型！不用再去那个下拉菜单里翻来翻去了，超级方便！虽然我也不太懂为什么有人会给模型起名叫 "final_v3_真的不改了"……

<span style="color: #ffc680"><strong>重要：</strong></span> 你得先放一个 [[Management Nodes/Derp Router|derpRouter]] 在流程里，不然它不理你。

### <span style="color: #80ffc0">能干什么</span>

<span style="color: #80aaff"><strong>刷新</strong></span>：你下载了新模型？点这个，不用重启 ComfyUI！重启多累啊。

<span style="color: #80aaff"><strong>移除</strong></span>：不想要哪个模型了，点一下它就没了。它不会生气的，应该吧。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：你可以把模型拖来拖去，虽然对出图没影响，但是看起来很整齐！我每次都排半天。

#### <span style="color: #80ffc0">系统面板选项</span>

<span style="color: #80aaff"><strong>显示文件夹名</strong></span>：要不要显示模型在哪个文件夹里。如果你的文件夹很乱的话……还是关了吧。

<span style="color: #80aaff"><strong>切换模型时清显存</strong></span>：换模型的时候自动把显存倒掉。我一般开着，因为我显存小，不清理的话会爆掉……

<span style="color: #80aaff"><strong>加载配置</strong></span>：可以把当前面板上放了哪些模型保存下来！这样你画人像用一套，画建筑用另一套，不用每次都重新加。默认是空的，因为每个人放的模型不一样嘛。
