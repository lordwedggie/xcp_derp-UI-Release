# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">VAE 加载器</span>

![[derpVaeLoader_01.jpg]]
放 VAE 文件的地方！点一下就能切。还有个神奇的功能——可以直接从模型文件里把 VAE 薅出来！这样你就不用单独去下载 VAE 了，我第一次知道的时候觉得好厉害。

<span style="color: #ffc680"><strong>重要：</strong></span> 你得先放一个 [[Management Nodes/Derp Router|derpRouter]] 在流程里，不然它不理你。

### <span style="color: #80ffc0">能干什么</span>

<span style="color: #80aaff"><strong>刷新</strong></span>：下了新 VAE？点一下，不用重启！重启好慢的。

<span style="color: #80aaff"><strong>移除</strong></span>：不想要了就删掉，很简单。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：可以拖来拖去，虽然我也不知道为什么要排，但能拖就拖嘛。

<span style="color: #80aaff"><strong>搜索</strong></span>：VAE 文件太多找不到？敲几个字就能搜到！我每次都记不住文件名，这个救了我好多次。

#### <span style="color: #80ffc0">系统面板选项</span>

<span style="color: #80aaff"><strong>显示文件夹名</strong></span>：要不要显示文件夹路径。我一般关掉，因为我的文件夹名字取得很随便……

<span style="color: #80aaff"><strong>从模型提取 VAE</strong></span>：开了以后不用单独指 VAE 文件夹，直接从模型文件里读！这样你就少了一个要管的文件夹，我老是忘记下 VAE，这个功能简直是救星。
