# <span style="color: #ff8080">老登</span> <span style="color: #ffffff">路由器</span>

![[derpRouter_01.jpg]]
超级重要的一个节点！它就像个信号中转站，让其他老登节点能互相传消息，不用拉线！没有它的话所有加载器都不会动的，所以一定要先放一个。

<span style="color: #ffc680"><strong>重要：</strong></span> 每次做新流程，第一件事就是把它放上去！不然其他加载器加多少都没用。

### <span style="color: #80ffc0">能干什么</span>

<span style="color: #80aaff"><strong>信号检测</strong></span>：自动找到流程里所有在发信号的节点，告诉你找到了几个、用了几个。我第一次看到的时候觉得好神奇。

<span style="color: #80aaff"><strong>添加信号</strong></span>：从找到的信号里挑你想要的加进来。能搜索！不用在几十个信号里翻。

<span style="color: #80aaff"><strong>拖拽排序</strong></span>：输出顺序可以拖来拖去！这个是真的有用——顺序不一样，节点收到的信号优先级也会变。

<span style="color: #80aaff"><strong>刷新注册表</strong></span>：新加了节点？点一下刷新就行，我以前不知道这个，每次都把路由器删了重新加……好傻。

<span style="color: #80aaff"><strong>跳转到节点</strong></span>：点一个信号条目，画布直接飞过去找那个节点！流程大了以后超级有用，不然满屏找节点眼睛要瞎了。

<span style="color: #80aaff"><strong>孤儿信号</strong></span>：如果你删了一个节点，它之前发的信号会留在这里一闪一闪的，提醒你清理。

#### <span style="color: #80ffc0">系统面板选项</span>

<span style="color: #80aaff"><strong>显示 ID</strong></span>：给每个信号标上编号。有两个重名节点的时候就能认出来了。

<span style="color: #80aaff"><strong>显示槽位名</strong></span>：要不要显示完整的槽位名字。关了更清爽。

<span style="color: #80aaff"><strong>显示信号类型</strong></span>：在每条后面标上它是 MODEL 还是 CLIP 还是 VAE……类型多了就知道谁是谁了。

<span style="color: #80aaff"><strong>显示虚拟连线</strong></span>：在路由器和连着的节点之间画一些虚拟的线。没啥实际用，但是画出来很好看！我喜欢开着。

<span style="color: #80aaff"><strong>隐藏连接槽</strong></span>：平时把连接槽藏起来，点中节点才显示。不然画布上太多东西看着好乱。

<span style="color: #80aaff"><strong>排序方式</strong></span>：按名字、类型还是 ID 排序，你喜欢怎么排就怎么排。
